\Charcoal\Admin\Ui

\Charcoal\Admin\UiActionContainerTrait

Provides methods for building groups of linked/action-capable items.

Can be used to build navigation bars, lists, and dropdowns. This trait {@todo should be replaced} with \Charcoal\Ui\Menu.
