\Charcoal\Admin\Action\Object

\Charcoal\Admin\Action\ObjectExportAction

Action: Export one or more objects from storage.

Required Parameters

  • obj_type (string) — The object type, as an identifier for a \Charcoal\Model\ModelInterface.

Optional Parameters

  • ident

Response

The response is in CSV mode.

  • success (boolean) — TRUE if the object(s) was/were exported, FALSE in case of any error.

HTTP Status Codes

  • 200 — Successful; Object(s) expoerted, if any
  • 400 — Client error; Invalid request data
  • 500 — Server error; Object(s) could not be expoerted, if any
