Class UpdateAction

Admin Update Action: Save an object in its Storage.

Required Parameters

  • obj_type string The object type, as an identifier for a ModelInterface.
  • obj_id mixed The object ID to load and update

Response

  • success boolean True if the object was properly saved, false in case of any error.

HTTP Codes

  • 200 in case of a successful login
  • 404 if any error occurs
