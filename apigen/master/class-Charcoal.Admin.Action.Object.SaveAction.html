Class SaveAction

Admin Save Action: Save an object in its Storage.

Required Parameters

  • obj_type

Response

  • success boolean True if the object was properly saved, false in case of any error.
  • obj_id mixed The created object ID, if any.
  • obj array The created object data.

HTTP Codes

  • 200 in case of a successful login
  • 404 if any error occurs
